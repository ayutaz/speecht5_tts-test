# speecht5_tts-test

[microsoft/speecht5_tts](https://huggingface.co/microsoft/speecht5_tts)を試すためのリポジトリ

# setup
```bash
uv venv -p 3.11
source venv/bin/activate
```

# install
```bash
uv pip install -r requirements.txt
```

# run
```bash
python transformers_modelling.py
```

or 

```bash
python tts_pipeline.py
```
